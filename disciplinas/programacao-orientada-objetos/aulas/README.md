## Como Adicionar uma Nova Aula

1. Crie um arquivo markdown seguindo o padrão de nomenclatura
2. Use o template abaixo:

```markdown
# [Data] - [Tema da Aula]

## Objetivos
- Objetivo 1
- Objetivo 2
- ...

## Conteúdo
### Tópico 1
...

### Tópico 2
...

## Exemplos
\```java
// Código exemplo
\```

## Exercícios
1. Exercício 1
2. Exercício 2

## Recursos Adicionais
- Link 1
- Link 2
```
