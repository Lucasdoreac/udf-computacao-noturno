Aula de Estrutura de Dados

13 de Fevereiro de 2025

Hoje tivemos nossa primeira aula de Estrutura de Dados com a Profa. Kadidja. Foi uma aula introdutória onde discutimos:

  • Como os dados são organizados na memória do computador
  • A importância das estruturas de dados para a programação
  • O impacto na performance dos programas

A professora ressaltou que precisaremos praticar muito e nos preparou para os desafios que virão.

Primeira Aula de POO

12 de Fevereiro de 2025

Segunda-feira começamos com POO em Java. O Prof. Gleidson explicou os conceitos fundamentais:

  • O que são classes e objetos
  • Como funciona o encapsulamento
  • A importância da orientação a objetos

Fizemos alguns exercícios práticos criando nossas primeiras classes em Java.
